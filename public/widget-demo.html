Step 1: Add data attributes to your hero elements

<section class="hero-section" data-vbb-hero>
  <img data-vbb-image src="default.jpg" alt="">
  <h1 data-vbb-headline>Default Headline</h1>
  <p data-vbb-subheadline>Default subheadline</p>
  <a data-vbb-cta href="/shop">Shop Now</a>
</section>

Step 2: Add the script tag

<script src="https://YOUR-DOMAIN.com/vbb-widget.js" async></script>

For local testing: <script src="/vbb-widget.js" async></script>

Step 3: Test with URL parameters

  • ?persona=USE_CASE_GAMING - Gamer
  • ?persona=BUDGET - Budget shopper
  • ?persona=COMPARE - Comparing
  • ?q=best+gaming+monitor - Search-style

Optional: Custom config

<script>
  window.VBB_CONFIG = {
    baseUrl: "https://your-cdn.com/assets/",
    registry: {
      images: { img_gaming: "/gaming.jpg", img_deal: "/deals.jpg" },
      ctas: { buy_now: { text: "Buy Now", link: "/checkout" } }
    }
  };
</script>
